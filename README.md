## About KANA

KANA is a Solana-native AI agent that orchestrates the flow of money with cold, calculated precision. This project showcases KANA's capabilities through an interactive portfolio website with real-time blockchain data monitoring.

## Features

- **ElizaOS Integration** - AI agent powered by ElizaOS framework
- **Real-time Token Monitoring** - Live data fetching from DexScreener API
- **Interactive Chat** - Direct integration with ElizaCloud for AI conversations
- **Video Showcase** - Interactive video presentation
- **Responsive Design** - Modern UI with Solana-themed styling
- **Server-Side Rendering** - Built with Astro for optimal performance

## Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Deployment

This project is configured for deployment on Vercel with web analytics enabled.

```bash
# Deploy to Vercel
vercel --prod
```
